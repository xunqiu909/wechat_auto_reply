# wxauto 自动回复监控 使用说明

## 运行环境

- Windows 10/11。
- 已安装并登录微信 PC 客户端。
- 启动前请保持微信主窗口可正常打开。

## 快速开始

1. 双击 `wxauto_auto_reply.exe` 启动程序。
2. 在右侧填写规则：
   - 聊天对象名：微信左侧会话列表里的名称，需要和微信显示一致。
   - 触发关键词：收到的消息包含该关键词时触发。
   - 自动回复内容：触发后发送的文本。
3. 点击“添加规则”。
4. 选择“检测一次”或“循环检测”。
5. 点击启动按钮。

## 多规则说明

- 可以添加多条规则，每条规则由“聊天对象 + 关键词 + 回复内容”组成。
- 启动后程序会先把多条规则涉及的会话打开一遍，让这些会话进入微信会话列表前方。
- 后续检测时会优先从会话列表点击切换，减少重复搜索，提高响应速度。

## 停止监控

- 在程序窗口中点击“停止监控”。
- 或在任意窗口按 `Ctrl + Shift + C` 强制停止循环检测。

## 回复记录

- 程序会在运行目录生成 `auto_reply_records.jsonl`。
- 记录内容包括：回复时间、聊天对象、触发规则、关键词、回复内容、原消息、发送是否成功。
- 程序界面底部“回复记录”页可查看、复制和清空记录。

## 注意事项

- 自动回复期间不要频繁手动切换微信窗口或编辑输入框。
- 聊天对象名称必须和微信显示一致，否则可能找不到会话。
- 如果杀毒软件提示未知程序，请选择信任；该程序为本地自动化工具，不会上传聊天内容。
- 如果微信版本升级导致无法定位会话，请重新启动微信和本程序后再试。
